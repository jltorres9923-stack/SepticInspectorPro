# Septic Inspector Pro — Legal & Support Site

Static GitHub Pages site: index, privacy policy, terms of use, support/FAQ.

## Deploy
1. Create a repo (e.g. `septic-inspector-pro-site`), push these files to the root.
2. Repo Settings → Pages → Source: `main` branch, `/ (root)` → Save.
3. Pages URLs for App Store Connect:
   - Privacy Policy URL: `https://<username>.github.io/<repo>/privacy.html`
   - Support URL:        `https://<username>.github.io/<repo>/support.html`
   - Marketing URL:      `https://<username>.github.io/<repo>/`
